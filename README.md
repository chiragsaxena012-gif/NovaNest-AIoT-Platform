# NovaNest-AIoT-Platform
AI-powered IoT smart home automation platform based on ESP32
Integrated with ESP RainMaker cloud for real-time monitoring & control
Uses Google Gemini AI for intelligent automation & smart decision-making
Supports multi-device control via mobile, laptop, voice assistants & manual switches
ESP32-CAM based face detection door system with security alerts
Real-time notifications using Telegram for critical events
Includes smart irrigation system for automated plant watering
Supports automated pet feeding system with scheduling & remote control
Features environmental monitoring (gas, temperature, humidity, motion, rain sensors)
Enables relay-based appliance automation (lights, fans, pumps, doors, etc.)
Designed as a low-cost, scalable & easy-to-install solution
Focused on safe, smart & energy-efficient living
